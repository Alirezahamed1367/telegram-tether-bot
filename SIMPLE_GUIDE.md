# 🎯 راهنمای خیلی ساده - برای مبتدی‌های کامل

## ❓ چرا نمی‌توانم ربات را به کانال `@tetherprice_toman` اضافه کنم؟

چون شما **ادمین** آن کانال نیستید! فقط یک عضو عادی هستید.

## ✅ راه‌حل چیست؟

یک **کانال میانی** می‌سازیم که:
- شما ادمین آن باشید
- ربات را به آن اضافه کنید
- پیام‌های کانال اصلی را به آن forward کنید
- ربات از آنجا بخواند

---

## 📝 چک‌لیست کامل (۱۰ قدم ساده)

### ✅ قدم ۱: ساخت ربات (۵ دقیقه)
```
□ به @BotFather پیام دهید
□ دستور /newbot بزنید
□ توکن ربات را کپی کنید و یادداشت کنید
```

---

### ✅ قدم ۲: ساخت کانال میانی - **مهم!** (۵ دقیقه)
```
□ در تلگرام: New Channel
□ نام: Tether Price Cache (یا هر نام دیگر)
□ نوع: Private Channel (خصوصی)
□ ربات را به این کانال اضافه کنید
□ ربات را ادمین کنید (با تمام دسترسی‌ها)
```

**چطوری شناسه کانال را پیدا کنم؟**
- به @userinfobot پیام دهید
- یک پیام از کانال میانی را forward کنید
- شناسه را کپی کنید (مثل: `-1001234567890`)

---

### ✅ قدم ۳: ساخت گروه برای دریافت پیام‌ها (۳ دقیقه)
```
□ در تلگرام: New Group
□ نام: Yuan Rate Updates (یا هر نام دیگر)
□ ربات را به گروه اضافه کنید
```

**چطوری شناسه گروه را پیدا کنم؟**
- به @userinfobot پیام دهید
- یک پیام از گروه را forward کنید
- شناسه را کپی کنید (مثل: `-1001234567890`)

---

### ✅ قدم ۴: ساخت حساب GitHub (۵ دقیقه)
```
□ به github.com بروید
□ Sign up کنید
□ ایمیل را تایید کنید
```

---

### ✅ قدم ۵: آپلود پروژه (۱۰ دقیقه)
```
□ در GitHub: New repository
□ نام: telegram-tether-bot
□ همه فایل‌های پروژه را آپلود کنید
  (خصوصاً پوشه .github/workflows)
```

---

### ✅ قدم ۶: تنظیم Secrets - **خیلی مهم!** (۵ دقیقه)
```
□ GitHub > Settings > Secrets > Actions
□ Secret اول:
  Name: BOT_TOKEN
  Value: [توکن ربات]
  
□ Secret دوم:
  Name: TARGET_GROUP_ID
  Value: [شناسه گروه، مثل: -1001234567890]
  
□ Secret سوم:
  Name: PRIVATE_CHANNEL_ID
  Value: [شناسه کانال میانی، مثل: -1001234567890]
```

⚠️ نام‌ها باید **دقیقاً** همین باشد (با حروف بزرگ)

---

### ✅ قدم ۷: فعال کردن GitHub Actions (۲ دقیقه)
```
□ GitHub > Actions
□ Enable workflows
```

---

### ✅ قدم ۸: تنظیم نرخ یوآن (۱ دقیقه)
```
□ در تلگرام به ربات پیام دهید:
  /setrate 7.12
```

---

### ✅ قدم ۹: Forward کردن پیام - **کلیدی!** (۲ دقیقه)
```
□ به کانال @tetherprice_toman بروید
□ آخرین پیام (که نرخ تتر دارد) را پیدا کنید
□ Forward کنید به کانال میانی خودتان
```

این کار را **هر روز** یا **هر بار که نرخ جدید آمد** تکرار کنید!

---

### ✅ قدم ۱۰: تست! (۵ دقیقه)
```
□ GitHub > Actions > Update Tether Rate
□ Run workflow
□ لاگ‌ها را چک کنید
□ در گروه تلگرام پیام را ببینید
```

## ❓ سوالات متداول

### چرا باید forward کنم؟
چون ربات نمی‌تواند از کانال عمومی بخواند. کانال میانی مثل یک "کپی" است.

### چند بار باید forward کنم؟
هر بار که کانال `@tetherprice_toman` نرخ جدید بگذارد.

### می‌توانم اتوماتیک شود؟
بله! با Telethon (راهنما در `ADVANCED.md`)

### اگر یک روز forward نکنم چی می‌شود؟
ربات از آخرین نرخی که forward شده استفاده می‌کند.

### هزینه دارد؟
خیر! کاملاً رایگان.
